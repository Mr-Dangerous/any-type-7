# Upgrade Relic System - Combinatorial Crafting

## System Overview

**Any-Type-7** uses a **combinatorial item crafting system** inspired by Teamfight Tactics. Players collect base Tier 1 items and combine them to create powerful Tier 2 relics with unique effects and stat bonuses.

**Core Formula**: `Tier 1 Item A + Tier 1 Item B = Tier 2 Relic C`

**Key Features**:
- 14 base Tier 1 items
- 105 unique Tier 2 combinations (14 base + 91 cross-combinations)
- Combined items inherit stat bonuses from both components
- Unique effects based on component synergy
- Potential for infinite scaling through further upgrades

## Infinite Scaling System

### Tier 3: The Ascendant Items

**Concept**: Combine a Tier 2 item with a matching Tier 1 component to ascend it to Tier 3

**Formula**: `Tier 2 Item + Tier 1 Component (used in T2) = Tier 3 Ascendant`

**Example**:
- **Overclocked Targeting** (Chronometer + Chronometer) + **Chronometer** = **Timeless Annihilation**
  - Stats: +50% Attack Speed, +5 Attack Range, +1 Projectile
  - Unique: Attacks fire an additional projectile

**Scaling Mechanic**:
- Tier 3 items cost exponentially more to upgrade (3x the resource cost)
- Only one Tier 3 item per ship (limitation for balance)
- Tier 3 items can be further upgraded with legacies for "+1" versions

---

### Tier 4: Ultimate Relic (Legacy Infusion)

**Concept**: Infuse any Tier 2 or Tier 3 item with all four legacy items to create Tier 4

**Formula**: `Tier 2/3 Item + Human + Alien + Machine + Toxic Legacy = Tier 4 Ultimate`

**Effects**:
- Item gains **all four legacy bonuses** (+50 Hull, +3 Hull Regen/sec, +50 Shields, +2 Energy Regen/sec)
- Stats increased by 50%
- Unique effect becomes AOE or doubled in potency
- Visual: Item glows with quad-colored aura (human/alien/machine/toxic themed)

**Example**:
- **Autocannon Protocol +4** (Tier 4 Ultimate)
  - +30% Attack Speed, +20 Damage, +50 Hull, +3 Hull Regen/sec, +50 Shields, +2 Energy Regen/sec
  - Every 3rd attack deals triple damage (improved from "every 5th deals double")
  - AOE: Triple damage attack hits 3x3 area

## Item Acquisition Sources

### Combat Rewards
- **Common drops**: Base stat items (Chronometer, Amplifier, Hull, Armor)
- **Uncommon drops**: Hybrid items (Thruster, Precision Lens, Aegis Plate)
- **Rare drops**: Ability items (Resonator, Capacitor, Dampener)
- **Boss drops**: Guaranteed Tier 1 item + 10% chance for Tier 2

### Sector Nodes
- **Mining Nodes**: 15% chance for Chronometer, Amplifier, Hull, or Armor
- **Outposts**: 20% chance for any base item
- **Graveyards**: 30% chance for defensive items (Aegis, Hull, Armor)
- **Traders**: Purchase any Tier 1 item (100 Metal or 50 Crystals)
- **Vaults**: Guaranteed Resonator, Dampener, or Legacy item (player choice)
- **Alien Colonies**: 2-3 random Tier 1 items on destruction

### Legacy Item Sources (Rare)
- **Mining Nodes**: 3% chance for random legacy item (very rare)
- **Vaults**: 50% chance for specific legacy (player sees before encounter)
- **Mothership Boss**: Guaranteed 1 of each legacy (Human, Alien, Machine, Toxic)
- **Trader**: 200 Metal + 200 Crystals each (expensive)

```gdscript
    # Sort alphabetically to handle A+B = B+A
    var sorted := [item_a, item_b]
    sorted.sort()

    var combo_key := sorted[0] + "+" + sorted[1]
    return combination_table.get(combo_key, "")
```

### Tier 3+ Scaling Formula

```gdscript
func calculate_tier_n_stats(base_item: Dictionary, tier: int) -> Dictionary:
    var scaling_factor := 1.0 + ((tier - 2) * 0.05)  # +5% per tier above 2

    var scaled_stats := base_item.duplicate()
    for stat in ["attack_speed", "attack_damage", "shields", "hull"]:
        scaled_stats[stat] = int(base_item[stat] * scaling_factor)

    return scaled_stats
```

## Balance Considerations

- **Tier 1**: Unlimited (stackable inventory)
- **Tier 2**: 6 equipped per ship (standard MOBA item limit)
- **Tier 3**: 1 per ship (powerful unique effect)
- **Tier 4+**: Upgrades to existing Tier 3 (doesn't use additional slot)

- **Tier 1 items**: 30% per combat encounter

## Conclusion

The **Upgrade Relic System** transforms item acquisition from linear progression into strategic crafting decisions. Players must choose between immediate power (Tier 2 combos) or long-term scaling (saving for Tier 3+), creating meaningful itemization choices every encounter.

**Design Pillars**:
1. **Discoverability**: Experiment with combinations to unlock recipes
2. **Strategic Depth**: Build synergies across 6 item slots
3. **Infinite Scaling**: Late-game resource sink with endless progression
4. **Visual Satisfaction**: Items visually evolve with power level

**Implementation Priority**:
1. Phase 1: All 14 Tier 1 base items + 14 same-component Tier 2 items
2. Phase 2: 40+ most synergistic cross-component Tier 2 items (legacy combinations)
3. Phase 3: Tier 3 Ascendant system
